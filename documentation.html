POST /acceptRequest.php

Accepts joining request.

Parameters

requestId: id of the request

POST /addBill.php

Adds a bill.

Parameters

payer: email of user that should pay the bill
description: description of bill
ammount: value of bill in pounds

POST /addUserToGroup.php

Sends join request to user.

Parameters

groupId: group's id
email: invited user's email

POST /changeGroupName.php

Changes the name of the group.

Parameters

groupId: group's id
name: new name for the group

POST /changeName.php

Changes the name of the group.

Parameters

name: new name

POST /changePass.php

Changes the password.

Parameters

password: old password
newPassword: new password

GET /confirmBill.php

Confirms the payment of the bill.

Parameters

billId: id of the bill

POST /createUser.php

Creates an account.

Parameters

email: email of the new user
password: password of the new user
name: name of the new user

POST /deleteRequest.php

Deletes request to join group.

Parameters

requestId: request's id

POST /deleteUserFromGroup.php

Removes user from group.

Parameters

groupId: group's id
email: removed user's email

GET /getBills.php

Returns list of bills.

GET /getEmail.php

Returns user's email.

GET /getGroups.php

Returns the groups the user is part of.

GET /getMoneyOwedToYou.php

Returns the total ammount of money other users owe to the user.

GET /getMoneyYouOwe.php

Returns the total ammount of money the user owes to other users.

GET /getMoneyOwedToYou.php

Returns the total ammount of money other users owe to the user.

GET /getRequests.php

Returns a list of group joining requests.

GET /getUsers.php

Returns a list users that are members of the group.

Parameters

groupId: group's id

GET /leaveGroup.php

Removes the user from the group.

Parameters

groupId: group's id

POST /login.php

Logs the user in.

Parameters

email: user's email
password: user's password

GET /logout.php

Logs the user out.
